TensorFlow: Smarter machine learning, for everyone

Thanks to machine learning, computers can do some amazing things these days, like instantly find all the pictures of your puppy in Google Photos. It’s pretty cool, but we think it’s just the beginning. So we created TensorFlow, an open source software library for machine learning that lets researchers, engineers, and enthusiasts exchange ideas quickly through code. And that, in turn, will accelerate research, making the technology work better in general — puppy shots, included.
