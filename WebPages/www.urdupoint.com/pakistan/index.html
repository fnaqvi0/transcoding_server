اُردو پوائنٹ پاکستان پر خوش آمدید

اُردو پوائنٹ کے نئے اور انتہائی منفرد سلسلے ’’اُردو پوائنٹ پاکستان‘‘ پر خوش آمدید۔ یہاں پاکستان کے ہر چھوٹے اور بڑے شہر کا علیحدہ سیکشن موجود ہے، جہاں اُس شہر کی تمام معلومات، خبریں، تصاویر، مضامین، ڈائیریکٹری ، ویڈیوز وغیرہ دستیاب ہیں۔ صارفین پچھلے دس سال کی تمام معلومات اب اپنے شہر کے صفحات پر دیکھ سکتے ہیں۔ اس نئے سلسلے کے آغاز کا مقصد صارفین کو اپنے شہر کی ہر بات بروقت پہنچانا ہے۔ اُردو پوائنٹ کے پاکستان بھر میں پھیلے نمائندے ہمارے ان صفحات کو بہتر بنانے میں دن رات مصروف ہیں۔ اگر آپ بھی اپنے شہر کی کوئی خبر، ویڈیو، تصاویر وغیرہ ہمیں اشاعت کیلئے ارسال کرنا چاہتے ہیں تو براہ مہربانی Pakistan@UrduPoint.com پر ای میل کیجئے۔ اپنے شہر کا نام ای میل میں ضرور ارسال کیجئے۔ ہمیں اُمید ہے ہے کہ آپ اس سلسلے کو ضرور پسند فرمائیں گے، اور اپنی قیمتی آراء سے ہمیں ضرور آگاہ کریں گے۔ اپنے شہر کاسیکشن دیکھنے کیلئے اُردو پوائنٹ۔کام/شہر کا نام درج کیجئے۔ یعنی UrduPoint.com/lahore, UrduPoint.com/karachi

ڈالر کی اونچی اڑان؛ پاکستان کو اسحاق ڈار کی یاد ستانے لگ گئی کیا نئے پاکستان سے بہتر پرانا پاکستان تھا؟ سوشل میڈیا صارفین کے دلچسپ تبصرے ( لاہور )

شہباز شریف نے نہ صبح دیکھی نہ شام ، دن رات کام کیا شہباز شریف نے خود کو بیمار کر لیا ،کینسر کے مرض میں مبتلا ہو گئے۔ نواز شریف کی صحافیوں سے غیر .. ( اسلام آباد )

ویسٹ مینجمنٹ کمپنی میں اربوں روپے کرپشن کا انکشاف کچرے کی صفائی کے نام پر قومی خزانے کی صفائی کا معاملہ قومی اسمبلی تک پہنچ گیا ( اسلام آباد )

آشیانہ ہاؤسنگ اسکینڈل کا آغاز کہاں سے ہوا ؟ آشیانہ ہاؤسنگ اسکینڈل میں چودھری نثار علی خان اور جنرل کیانی کا کیا کردار ہے؟ معروف صحافی نے انکشاف .. ( لاہور )

بیگم کلثوم نواز کے انتقال کے بعد شریف خاندان کو ایک اور صدمہ مسلم لیگ ن کے رہنما عابد شیر علی کی والدہ انتقال کر گئیں ( لاہور )

میں نے سُنا ہے کہ ڈالر 140 تک جائے گا پاکستان تحریک انصاف کے وفاقی وزیر نے ڈالر کی قدر میں مزید اضافے کا عندیہ دے دیا ( کراچی )

ڈسٹرکٹ جیل کیمپ آشیانہ ہاؤسنگ اسکیم میں ملوث سرکاری افسران سے بھر گئی شہباز شریف کو نیب کی تحقیقات کے بعد کوٹ لکھپت جیل منتقل کرنے کا فیصلہ،بیڈ .. ( لاہور )

نیب کی زیر حراست شہباز شریف دو روز سے سو نہیں سکے حوالات میں قید ہونے کے بعد شہباز شریف کی طبیعت خراب ہو گئی ( لاہور )

وزیر اعلیٰ پنجاب عثمان بزدار کے خلاف میڈیا کی طرف سے منفی مہم کا انکشاف ڈیفنس میں ایک گھر ہے جہاں صحافیوں کو بلا کر عثمان بزدار کے خلاف ہدایات .. ( لاہور )

نیب نے صدر مسلم لیگ ن شہباز شریف کے خلاف اہم ثبوت حاصل کر لیے ( لاہور )

شہباز شریف اپنی گرفتاری سے انتہائی دلبر داشتہ ہیں شہباز شریف سلاخوں کے پیچھے دیواروں کو گھورتے رہتے ہیں ( لاہور )

وزراء کے بے مقصد چھاپوں اور احتساب میں جلد بازی نے ملک کا 2800ارب روپے کا نقصان کر دیا عمران خان نے بطور وزیر اعظم حلف اٹھایا تھا تو ملک میں غیر .. ( لاہور )

روپے کی قیمت میں کمی ، ڈالر کی قیمت میں اضافہ ہو گیا انٹربنک مارکیٹ میں ڈالر کی قیمت 137 روپے ہو گئی، فاریکس ڈیلرز ( کراچی )

لاہور کی صفائی کا ٹھیکہ غیر شفا ف تھا، منصوبے کا آڈٹ کروائیں گے،سابقہ پنجاب حکومت صرف کوڑا اٹھانے کیلئے سالانہ 18 ارب روپے دیتی رہی، ماضی .. ( اسلام آباد )

خواجہ آصف نے ایک بار پھر تحریک انصاف کو'شرم اور حیا' یاد دلا دی خود کشی کر لو لیکن اس کے لیے بھی شرم اور حیا کی ضرورت ہوتی ہے،حکومت کے آئی ایم .. ( لاہور )

14سالہ لڑکی بھائی اور ماموں کی طرف سے جنسی زیادتی کا شکار ہونے کے بعد حاملہ ہو گئی لڑکی کی والدہ اسے غیر اخلاقی کام کرنے پر اکساتی تھی، مقدمہ .. ( پشاور )

نیشنل سکیورٹی موجودہ حکومت کی اولین ترجیح ہے جس پر کسی قسم کا کوئی سمجھوتہ نہیں کیا جائے گا، پہلے حکمرانوں نے اس ملک کے ساتھ بہت ظلم کیا ہے وزیر .. ( اسلام آباد )

عمران خان ملک کے وزیراعظم ہیںیا سینیٹری سپروائزر، ایسا لگتا ہے انہوں نے اسی کام کے لئے سیاسی لوٹے جمع کر رکھے ہیں، سعید غنی ( اسلام آباد )

ماضی کے رجحانات سے ہٹ کر ہم اچھی طرز حکمرانی کے ذریعے ترقی یافتہ بلوچستان کی بنیاد رکھنے جارہے ہیں وزیراعلیٰ بلوچستان جام کمال خان کی میڈیا .. ( کوئٹہ )
